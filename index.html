(Submitted) HeatSurf:Thermal Cues Enable Temperature-Aware dexterous Manipulation
Yiping Lu*, Caiyun Liu, Jiawei Xia, Haozhe Lou, Yongliang Shi†
CVPR, 2026
[Paper] [Video]

Kitchen robotic manipulation often involves thermal interactions, but mainstream RGB-D-based robot methods ignore this key thermal modality. To fix this, we present HeatSurf—a plug-and-play module that gives robots surface-based thermal perception: it adds a thermal sensor to an RGB-D camera, maps per-point thermal intensity/gradients to a Gaussian Splatting scene representation, and integrates this into an imitation learning policy. We tested it on 3 real-world temperature-sensitive kitchen tasks (pick hotter item, spread oil evenly, clean hot spills).

Robust Online Calibration for UWB-Aided Visual-Inertial Navigation with Bias Correction
Yizhi Zhou*, Jie Xu, Jiawei Xia, Zechen Hu, Xuan Wang†
IROS, 2025
[Paper] [Video]

This paper proposes a robust online calibration method for UWB anchors in Visual-Inertial Navigation Systems (VINS). It overcomes issues with existing methods, such as reliance on accurate robot localization and sensitivity to initial anchor positions. Based on MSCKF, our approach uses stochastic optimization for reliable initialization and a Schmidt Kalman Filter for online refinement. Simulations and real-world tests show improved accuracy and robustness.

CVIRO: A Consistent and Tightly-Coupled Visual-Inertial-Ranging Odometry on Lie Groups
Yizhi Zhou*, Ziwei Kang, Jiawei Xia, Xuan Wang†
IROS, 2025
[Paper] [Video]

This paper proposes a consistent and tightly-coupled visual-inertial-ranging odometry (CVIRO) system based on the Lie group. Our method incorporates the UWB anchor state into the system state, explicitly accounting for UWB calibration uncertainty and enabling the joint and consistent estimation of both robot and anchor states. Further more, observability consistency is ensured by leveraging the invariant error properties of the Lie group.

Point-line-Thermal-LiDAR-Inertial SLAM system with pre-processing of thermal images
Jiawei Xia*, Yixiao Feng*

We propose PL-LITO, a robust LiDAR-Thermal-Inertial SLAM system for industrial safety. Addressing thermal image instability (AGC) and low contrast, it integrates online photometric calibration to enhance feature tracking. Leveraging thermal images’ strong structure (weak texture), it uses a deep learning point-line feature network and tightly-coupled IESKF, fusing multi-modal data with LiDAR depth for high-precision low-light localization. With photometrically calibrated 8-bit data, it builds a Probabilistic Intensity Voxel Map for consistent thermal monitoring and real-time anomaly detection. Model inference is accelerated via TensorRT. A custom dataset (challenging environments) was collected using a self-developed 3D-printed device (Mid360 LiDAR, F460 Thermal Camera).

Robocup@Home: Develop an intelligent home service robot based on ros
Jiawei Xia*, Xianglei Dong, Wengcong Zhang, Boyi Zhang, Yutong Sun

Developed an intelligent home service robot using Kinova Gen2 and Kinect Azure camera based on ROS Noetic, with functions including guest reception and guidance, voice interaction, object recognition, and room cleaning. Used Graspnet algorithm to get the pose of the object, which was sent to the end of the robotic arm by TF listener. Controlled the arm using Actionlib and MoveIt!, and used the RRT* algorithm for efficient trajectory generation. Adopted Azure AI Speech SDK for speech recognition and voice output, and combined it with a finite state machine to enable simple human-computer voice interaction. For object recognition, I integrated YOLOv5 and Kinect DK. Applied Cartographer for 2D indoor mapping, used IRIS LaMa for robot localization, and achieved local path planning based on the DWA and global path planning based on the A* algorithm.

RoboMaster University League 1v1 Combat Competition
Yutao Guo*, Hanhong Fu, Jiawei Xia

Used the open-source vision algorithm framework of the South China Normal University - PIONEER team to our own robot, used PnP to calculate the 3D position of the armor plate, and used Kalman filtering for tracking; The robot used four M3508 motors for the chassis and two M6020 motors for the turret, controlled via CAN communication and PID algorithms for current, speed, and position loops. A six-axis gyroscope provided data via SPI, fused with the Mahony algorithm to compute quaternion-based turret orientation. Used feedforward PID to improve the gimbal drive and achieve gravity compensation.
